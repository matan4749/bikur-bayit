# ביקור בית ליולדת

אתר מידע וקביעת ביקור בית ליולדות — שירותי בריאות הציבור (טיפת חלב).

## תכונות

- מסך פתיחה עם מידע על זכאות לביקור בית
- טופס קביעת ביקור (דמו)
- מידע חשוב לחודש הראשון (הנקה, צהבת, חיתולים, משקל, סימני אזהרה)
- סרטוני הדרכה ושאלות נפוצות
- מעקב יומי עם התראות
- יצירת קשר עם טיפת חלב ויועצת הנקה

## הרצה מקומית

```bash
cd bikur-bayit
python3 -m http.server 8765
```

פתחו בדפדפן: http://localhost:8765
